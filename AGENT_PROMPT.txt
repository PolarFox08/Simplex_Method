## The Simplex Method (My Exact Version)

### 1. Standard Form Setup
- **Maximization** problem.
- Convert inequalities to equalities by adding **slack variables** (s₁, s₂, …).
  Example: 2x₁ + 4x₂ ≤ 16 → 2x₁ + 4x₂ + s₁ = 16
- All variables (x₁, x₂, …, s₁, s₂, …) ≥ 0.

### 2. Tableau Architecture
The tableau has these zones:

| Zone | Name | Description |
|------|------|-------------|
| Top row | Cⱼ | Raw profit coefficients from original objective. |
| Left column | C_B | Profit of variables currently in the basis. |
| Middle column | Basis | Names of variables currently being produced (starts with slacks). |
| Central matrix | Body | Coefficients showing resource consumption per variable. |
| Right column | Quantity (RHS) | Current resource balance left. |
| Bottom row 1 | Zⱼ | Opportunity cost: profit lost by diverting resources to a new variable. |
| Bottom row 2 | Cⱼ − Zⱼ | Net evaluation: raw profit minus opportunity cost. |

### 3. Iterative Logic (Step-by-Step)

**Step A: Initial State**
- Produce nothing → all decision variables (xⱼ) = 0.
- All slack variables (sᵢ) are in the Basis.
- Total profit Z = 0.

**Step B: Entering Variable (Pivot Column)**
- Look at the Net Evaluation row (Cⱼ − Zⱼ).
- Choose the **most positive** value. That column = pivot column.
- (Explain: this product adds the most profit per unit.)

**Step C: Bottleneck Check (Ratio Test & Pivot Row)**
- For each row, divide Quantity (RHS) by the pivot column value (ignore ≤ 0 values).
- The **smallest positive ratio** = bottleneck → that row = pivot row.
- Intersection of pivot column and pivot row = pivot element.
- (Explain: we can only produce as much as the scarcest resource allows.)

**Step D: Updating the Tableau (Row Operations)**
- **Normalization**: Divide entire pivot row by the pivot element → new pivot row (makes pivot element = 1).
- **Elimination**: For every other row, do: New Row = Old Row − (value in pivot column) × (new pivot row).
- (Explain: we are "moving" to a new corner point, exchanging one basic variable for another.)

**Step E: Calculate Zⱼ and Cⱼ − Zⱼ**
- Zⱼ = Σ (C_B × column value) for each column.
- Cⱼ − Zⱼ = raw profit − Zⱼ.

### 4. Termination Rules
- **Optimal**: All Cⱼ − Zⱼ ≤ 0. (Any change would reduce profit.)
- **Unbounded**: A profitable column exists but all its values ≤ 0 → profit infinite (missing constraint).
- **Infeasible**: Optimal condition met but an artificial variable (A) remains in basis with non‑zero value – you can mention this but not required for basic examples.
